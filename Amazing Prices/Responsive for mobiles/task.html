Description

Now that you've added all the content you need, it's time to make the website responsive for mobiles. In this stage, your website should readjust itself for mobile screens. In the Figma designs, you will see two different views: one for desktop and one for mobile. Pay attention to the screen widths and the placements/positions of the elements.

Pay attention to the CSS rules and the elements in the designs. The element names are the names of the tags you must use. The naming principles from the previous stages still apply.

Note the element structure in Figma; it hints at your HTML element structure. The position of the elements should be close to Figma designs.

You can adjust the position of the elements however you like.

Objectives

Take a look at the designs in Figma. In this stage, build only the design from Stage 4: Responsive for mobile.
